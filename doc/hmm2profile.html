NAME

hmm2profile.pl - Creating a phylogenetic profile out of a HMMSEARCH result.

SYNOPSIS

hmm2profile.pl [options]

DESCRIPTION

This script search a fasta database of sequences by a HMM. It then parses the resulting file to create a phylogenetic profile.

INSTALLATION

You need BioPerl and Malay's SeqToolBox to run this script. Follow BioPerl's installation for installing it. Contact Malay (mbasu@jcvi.org) for a copy of SeqToolBox. Unzip the modules in your PERL5LIB. You need to first set up the NCBI taxonomy database for using this script. For this set up an environment variable "SEQTOOLBOXDB" pointing to a directory that can store about ~1.5 GB of data. After setting up this enviornment variable, run install/update_taxonomy.pl script. This will create all the databases required for running this script.

OPTIONS

--help | -h

Print this help message.

--file | -f

The full path of HMM file.

--db | -d

The multifasta file containing the sequences.

--options| -o

Options directly passed through HMMSEARCH. Pass this as complete string like this "-E 0.001 --threads 4". Keep the quote around the options.

--taxdis | -t

A taxnomic distribution file. Should have been created using taxdist_fasta.pl. These taxons are appended to the final profile files, after checking the presence or absence of hits in these taxa.

--hmmer | -H = hmmer_result_file

You can create a profile out of a pre-existing HMMER search. You can not use both -f,-H, or -a at the same time.

--add | -a

You can add taxon data to a pre-existing result. Use -t with this option.

--clean

Clean the HMMER result file.

--cutoff | -c = bit_score_cutoff

You can use a bit score cutoff while parsing a preexisting HMMER result file.
